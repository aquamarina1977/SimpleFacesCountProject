# SimpleFacesCountProject
Проект из области компьютерного зрения. Этот код определяет количество лиц на фотографии и является вариацией многочисленных проектов в интернете для определения человеческих лиц или количества лиц. Файл haarcascade_frontalface_default.xml взят из репозитория на GitHub, размещённом по адресу: https://github.com/opencv/opencv.git
